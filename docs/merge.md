# isocall merge

The `merge` command combines one or more transcriptome profiles into one
merged profile for downstream isoform calling. Inputs can be single-sample
profiles generated by `isocall profile`, or profiles that were already
produced by `isocall merge`.

## Basic usage

```bash
isocall merge --profiles sampleA.gz sampleB.gz sampleC.gz --output merged.gz
```

## Required arguments

| Argument | Description |
|----------|-------------|
| `--profiles` | One or more profile files produced by `isocall profile` or `isocall merge` |
| `--profile-list` | Path to a file listing profile paths, one per line |
| `--output` | Output path for merged profile (must end with `.gz`; written as BGZF) |

Provide exactly one of `--profiles` or `--profile-list`.

## Optional arguments

| Argument | Default | Description |
|----------|---------|-------------|
| `--io-threads` | 2 | Number of BGZF compression/decompression threads to use during merge |

## Input requirements and validation

- Input profiles must have consistent chromosome lengths. If the same
  chromosome has different lengths across inputs, merge fails.
- `--output` must end with `.gz`.
- Sample order is preserved from the `--profiles` argument list.
- `--io-threads` defaults to `2` because `1` disables the HTSlib thread pool
  entirely, while `2` enables parallel BGZF IO with modest CPU overhead.
- BGZF inputs can use parallel decompression; plain gzip inputs are still
  accepted but are read single-threaded.

## What the command does

1. Reads all input profiles and tracks each sample independently.
2. Merges compatible read bundles across inputs into a unified profile.
3. Preserves per-sample read counts for downstream commands.

## Output format

The merged profile is a BGZF-compressed, gzip-compatible file with:

1. Header section:
   - One `#ID` line per sample
   - `#CR` lines with chromosome names and lengths
2. Body:
   - Encoded merged read bundles used by `isocall call`

## Next step

Use the merged profile with `isocall call` and the database generated by
`isocall prep-isoforms`:

```bash
isocall call \
    --merged-profile merged.gz \
    --known-isoforms ref_seq.isoforms.gz \
    --reference hg38.fa \
    --output-prefix results
```
